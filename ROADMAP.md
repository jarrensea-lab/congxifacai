## v7.5.0-dev：盈利策略管线重构

目标：让主报告和标的池直接服务“今天/明天怎么操作”。所有研究、数据、辩论和报告都必须转化为账户可执行判断、标的池状态、预警条件和复盘记录。

### P0

- [x] 主报告第一屏改为账户动作策略。
- [x] 标的池分层为 `executable` / `watching` / `research_reference` / `removed`。
- [x] Sentinel/Serenity 研究候选默认进入研究参照，不绕过评分直接买入。
- [x] 新增单标的 Target Snapshot，归一行情、K 线、资金流、北向、新闻、财务和研究证据。
